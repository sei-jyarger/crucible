Crucible: A Cyber Simulation Framework

Crucible delivers virtual environments featuring a modeled topology, simulated user activity, and scenario events in order to improve performance of mission-essential tasks or individual qualification requirements.


● Open-source cyber-simulation application-framework
● Modular design with extensive APIs
● Customizable, immersive, browser-based user interface
● “Infrastructure as code” topology building
● Flexible integration of third-party open-source tools
● Scenario-based simulation and performance assessment
● On-prem and cloud
● Automation
● Open standards

The Framework


Crucible applications provide simulation developers with tools to develop high quality cyber security training events.

Each Crucible application has a robust API that encourages interoperability.


Crucible Toolset


● Crucible facilitates training development by integrating with industry-leading software.

● Numerous infrastructure providers supported -- both on-prem or in the cloud.

● Keycloak supported for identity and access management.
